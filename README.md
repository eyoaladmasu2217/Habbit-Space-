# Habit Space

Habit Space is a modern, interactive habit tracker built with React Native and Expo. It features a unique "Space" concept where each habit has its own dedicated, zoomable dashboard for tracking progress.

## ‚ú® Features

- **Interactive Zoomable UI**: Each habit is a card that you can "zoom into" using pinch gestures or tapping.
- **Daily Check-ins**: Easily track your daily progress with a single tap.
- **Customizable Habits**: Add habits with personalized titles, icons, and colors.
- **Persistence**: Your data is saved locally on your device using AsyncStorage.
- **Fluid Animations**: Smooth transitions and gestures powered by React Native Reanimated.
- **Dark Mode UI**: A sleek, dark-themed interface designed for focus.

## üöÄ Getting Started

### Prerequisites

- Node.js
- npm or yarn
- Expo Go app on your mobile device (to preview)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/eyoaladmasu2217/Habbit-Space-.git
   cd Habbit-Space-
   ```

2. Navigate to the app directory:
   ```bash
   cd temp_app_2
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npx expo start
   ```

5. Scan the QR code with your Expo Go app (Android) or Camera app (iOS).

## üì∏ Usage

- **Tap** a habit card to zoom into its Space.
- **Pinch In** while inside a Space to zoom back out to the main grid.
- **Long Press** a card on the main grid for a quick hint.
- **Tap the "+" button** to create a new habit Space.
- **Tap the Trash icon** inside a Space to remove that habit.
